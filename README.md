## Build & Run

### If using Maven

```bash
mvn clean package
mvn test
mvn exec:java
```

### If using Gradle

```bash
./gradlew build
./gradlew test
./gradlew run
```

### If using plain Java

```bash
javac -d out src/**/*.java
java -cp out Main
```
